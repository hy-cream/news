# new这是一个用gank.io接口实现的技术干货App
实现了几个功能点：
1.浏览界面包括Android，ios,瞎推荐，妹子图
2.点击浏览各个页面，并且可以刷新
3.妹子图的下载
4.自定义轮播图

UI布局部分遵循MD的设计模式，主要用recycleView结合AppBarLayout和卡片式布局来实现的
网络框架部分主要采用retrofit2.0框架
下载暂时只实现了单线程断点下载，未完待续
